# ERP System

Enterprise Resource Planning (ERP) software to plan and manage daily activities such as supply chain, accounting, and sales. It can also be used to automate and simplify individual activities across the business.

This project is maintained by **daniellopez882**.

## Installation

### Prerequisites
- Python 3.8 or later
- PostgreSQL 11.0 or later

### Install Dependencies
To install the required Python packages, run:

```sh
pip3 install -r requirements.txt
```

### Running the Server
You can start the ERP server using the command-line interface:

```sh
./erp-bin
```
